User Routes

Method Path Description
POST /register Register a new user.
POST /users Login a user that's already registered.
GET /users To get all the users that are registered
GET /users/:id Get a user by using the unique id.
DELETE /users/:id Delete a restaurant by using it's unique id.
PUT /users/:id Update an existing user profile by using the the id.

Restaurants routes

Method Path Description
GET /items Get all the restaurants.
GET /items/:id Get a single restaurant by using the unique id.
POST /items Add a restaurant to the API.
PUT /items/:id Update a restaurant by making use of the unique id.
DELETE /items/:id Delete a single restaurant by using the id.

Cart routes

Method Path Description
POST /user/:id/cart Add an item to the Cart.
GET /user/:id/carts Get a users Cart.
PUT /user/:id/cart/:id Update a Cart item
DELETE /user/:id/cart/:id Delete a restaurant from the cart by making use of the unique id.
DELETE /user/:id/cart/ Clear the cart.
